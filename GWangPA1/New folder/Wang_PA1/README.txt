HOW TO RUN PROGRAM

Eclipse

The program can be ran in Eclopse by importing the parent folder as project and run the main class
Main class: GWangPA1
(with or without command line arguement as mentioned below)

JDK 6
The source code can be compiled with JDK 6 from the command line with or without argument as below
With Arguement: java GWangPA1 [inputFile] [outputFile]
Without Argument: java GWangPA1

Note: There would be prompt in the console asking for input and output file locations if you command line arguments are complete
